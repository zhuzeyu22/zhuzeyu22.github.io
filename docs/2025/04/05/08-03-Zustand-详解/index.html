Zustand 详解

Zustand 是一款轻量级、无依赖的 React 状态管理库，主打简洁和极致的开发体验。

核心特性

  • 轻量、无依赖、Hooks API
  • 支持中间件、可组合 store
  • 性能优秀、易于维护
