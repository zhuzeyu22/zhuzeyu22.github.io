Next.js 详解

Next.js 是一个基于 React 的服务端渲染（SSR）和静态站点生成（SSG）的框架，支持全栈开发和高效的页面路由。

核心特性

  • 文件即路由、API 路由
  • SSR、SSG、ISR
  • 内置 CSS/JS 支持、自动代码分割
  • 优秀的 SEO 和性能

基本用法

适用场景

  • 企业级应用、内容型网站、SEO 友好的项目
