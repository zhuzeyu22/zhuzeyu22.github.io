# 模块化

使用 yarn 配置 workspace

```
e.g.
├───modules
│ ├───memo-a
│ └───memo-b
```

- 安装某个模块的依赖

  yarn workspace memo-a add ./modules/memo-b

- 安装到工作空间

  yarn add cross-env -W

---

yarn install 时候会自动构建链接
