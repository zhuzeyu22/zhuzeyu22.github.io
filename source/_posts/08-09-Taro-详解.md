# Taro 详解

Taro 是一套遵循 React/Vue 语法规范，多端统一开发的框架，支持小程序、H5、RN 等多端。

## 核心特性

- 一套代码多端编译
- 支持 React/Vue/小程序/H5
- 丰富的生态与插件

## 基本用法

```js
import Taro from '@tarojs/taro'
import { View } from '@tarojs/components'
export default function Index() {
  return <View>Hello Taro</View>
}
```

## 适用场景

- 多平台小程序、H5 项目统一开发
