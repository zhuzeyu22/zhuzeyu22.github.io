## 推荐工具：Husky

[Husky](https://github.com/typicode/husky) 是一款流行的 Git hooks 工具，可用于在代码提交、推送等阶段自动执行脚本，如代码检查、格式化、测试等。  
它常与 ESLint、Prettier、Commitlint、lint-staged 等工具配合，保障团队代码质量和开发流程自动化。
