Vuex 是专为 Vue.js 应用程序开发的状态管理库。

## 核心概念

- State、Getter、Mutation、Action、Module
- 响应式状态树、集中式管理
- DevTools 支持

## 基本用法

```js
const store = new Vuex.Store({
  state: { count: 0 },
  mutations: {
    increment(state) { state.count++ }
  }
})
```

## 适用场景

- 复杂 Vue 项目状态管理
- 需要持久化/模块化状态管理
